Hera

Team: Yiokling Chu, Manuel Garza, Kyle Bittner.

Hera is a pill reminder app I made in collaboration with a team. HERA keeps people on track of their medication and prescriptions with daily reminders. This app helps people that tend to neglect doctor's orders. Hera assists people to manage to keep their number of medications straight.

Some of Hera's functionality are:

  • HERA allows you to create any type of recurring reminders (daily, weekly, monthly, every X days).
  • HERA tracks the remaining quantity of each medication and shows a refill alert when running low.
  • HERA provides missed medication alerts.

APP PROCESS:

PROTOTYPE:

Hera's first prototype was made using Figma, we tried to add all the functionalities and basic layout closest to want we wanted the final product to look like.

DEVELOPMENT:

In order to develop Hera, we used React Native, and code with JavaScript.


FINAL APP:


Hera is linked to a Firebase database. It has the user input a name for the medication, a recurrence (time and if daily/weekly/monthly), and for how long. Then, the algorithm sets a reminder for the specific medication which holds the recurrence and time limit.

There are two main parts for Hera. Hera is the front-end react-native app and Ares is a redux-thunk middleware application that holds functions and database calls.
