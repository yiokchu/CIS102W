PROGRAM: ONLINE PHONE BOOK


This is a network application that manages an online phone directory. In this program, the phone book's data model is saved in a file on the server's computer. The customers are able to look up a phone number or add a name+phone number to the directory. In addition, the server is able to handle multiple clients with all clients having access to the same phone book. This object is created by the server at startup and passed to the client handlers.

The program runs the following way: the client sends a connection request to the server, the server creates a socket and binds it to the address (host, port). The port numbers are the same (5000) therefore they connect, if they were different, it would give an error.
The finding and adding a personâ€™s number or person's name works since the methods send the server the coded name or coded name+phone number. Then, the server class calls the PhoneClientHandler to add or find the person. The handler decodes the data (message), and sends a reply if the number was found or not (using the get/add methods defined in phonebook), or if the name and number were added to the client (coded). Finally, the reply (message) is displayed to the client by the phonebookclient. Then to disconnect, the client closes the connection with the server (If the client does not connect, the server just waits for connection).
